#Onepagetemplate

A simple template made using Reactstrap and mdbreact

Steps to Install:

1. Go to the project folder install 'npm install -g expo-cli'
2. Go to the project folder install Bootstrap 
   'npm install --save bootstrap'
   'npm install --save reactstrap react react-dom'
3. Adding Bootstrap css in the App.js file
   import 'bootstrap/dist/css/bootstrap.min.css';
4. Go to the project folder install ' npm install --save mdbreact'
5. Adding Bootstrap css in the App.js file
   import '@fortawesome/fontawesome-free/css/all.min.css'; 
   import 'bootstrap-css-only/css/bootstrap.min.css'; 
   import 'mdbreact/dist/css/mdb.css';
   
6. Run npm start: this will create a localhost go to page select run in web browser.

Features:

1. Responsive UI: For desktop viewports and for mobile like viewports.
2. Supports last stable version of chrome/firefox/safari/edge for simplicity, can be modified though!
3. Good and consistent UI.
